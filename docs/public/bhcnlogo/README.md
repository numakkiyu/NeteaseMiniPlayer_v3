# BHCN STUDIO Logo 资源

本目录存放文档站页脚使用的 BHCN STUDIO 标识文件。

## 文件说明

```txt
bhcn-logo-full-light.svg    # 适合浅色背景的完整标识
bhcn-logo-full-dark.svg     # 适合深色背景的完整标识
bhcn-logo-symbol-light.svg  # 适合浅色背景的图形标识
bhcn-logo-symbol-dark.svg   # 适合深色背景的图形标识
```

目录中也保留了 PNG 导出文件，方便在不适合使用 SVG 的环境中备用。文档站默认使用 SVG 文件，并根据 VitePress 的明亮 / 暗黑模式自动切换。

## 页脚文案

站点页脚显示：

```txt
本项目由 BHCN STUDIO 维护
```
