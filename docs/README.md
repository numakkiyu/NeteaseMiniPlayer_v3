# Documentation

This directory contains public documentation for users, developers, and contributors.

## Product Guides

- [NMPv3](nmpv3/README.md): lightweight single-JS NetEase Cloud Music player standardized from the v2.5 UI and interaction model.
- [NMPv3+](nmpv3-plus/README.md): advanced framework and extension ecosystem.

## Install

NMPv3 can be installed with either npm or pnpm:

```bash
npm install @netease-mini-player/v3
pnpm add @netease-mini-player/v3
```

NMPv3+ can also be installed with either npm or pnpm:

```bash
npm install @netease-mini-player/v3-plus
pnpm add @netease-mini-player/v3-plus
```
